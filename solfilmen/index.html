Jag och Emelie pratade om att solen går upp så pass mycket senare och ner tidigare här på Island när vi kom hit för ett tag sedan. Vi kom då in på det vi har i Sverige för att hålla koll på när solen går upp och ner: Solfilmen. Jag tror de flesta (åtminstone äldre) vet vad Solfilmen är så enda förklaringen blir ett klipp:

Här på Island tror jag inte att dem har tillgång till denna lyx, att enkelt en gång i veckan avgöra om solen går upp och ner senare eller tidigare. Så jag bestämde mig för att göra min egen, helt automatiserade, solfilm.

Utan att gå in på allt för många tekniska detaljer tog det mig ett antal timmar att knåpa ihop en solfilms-generator. Det var naturligtvis inte bara för att jag saknade solfilmen så mycket som jag gjorde detta. Det krävdes ett par väldigt intressanta tekniker för att åstadkomma denna generator, så det var roligt att läsa på om tekniker jag normalt sätt inte använder (så mycket). Jag kan lista de olika teknikerna jag tycker karaktäriserar projektet, även om ni som läser kanske inte är så insatta:

  • FFMPEG - Teknik för att klippa/rendera film programmatiskt (aldrig använt förut)
  • ImageMagick - Teknik för att generera bilder programmatiskt (använt i ett projekt förut)
  • Amazon Web Services - Platform att "förvara" koden och de genererade filmerna (använder det varje dag i jobbet)
  • Docker - Teknik för att köra mitt program på vilket operativsystem som helst, bara Docker är installerat (använder det varje dag i jobbet)
  • Python - Programmeringsspråk (använder det sällan, men är nog ett av alla språk jag har testat jag är mer bekväm med)
  • timeanddate.com - Webbsida med data på soluppgång och nergång för bland annat Island.

Tanken är nu att 1 solfilm i veckan ska genereras! Om du vill kan du se dem genom att klicka här! Får se om jag orkar göra några uppdateringar eller om det blir ett liknande klipp varje vecka. Kul litet helgprojekt oavsett.
